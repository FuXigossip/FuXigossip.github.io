云计算实验

山东大学 计算机科学与技术 学院
云计算技术 课程实验报告
学号：202200130052 姓名：王旭 班级： 22级数据
实验题目：利用云平台搭建个人博客
实验学时：2 实验日期： 3.19
实验目的：本实验旨在通过实践操作，掌握基于云计算环境的网站部署技能。
具体包括：构建环境并实现个人博客/网站的搭建，撰写实验报告，并将这份报告作为一篇博文发布在自己的博客 / 网站上。
硬件环境：
联网的计算机一台
软件环境：
Windows or Linux
实验步骤与内容：
一、环境准备
基础工具安装
安装Git（Windows用户下载安装包默认配置安装）
git –version # 验证安装
安装Node.js（需≥v12.0）
node -v # 验证安装
npm -v # 验证包管理器
Hexo框架安装
npm install hexo-cli -g # 全局安装Hexo
hexo -v # 验证安装
二、GitHub配置
SSH密钥配置
ssh-keygen -t rsa -C “your_email@example.com“ # 生成密钥
cat ~/.ssh/id_rsa.pub # 复制公钥内容
登录GitHub > Settings > SSH and GPG keys > New SSH key添加
仓库创建规范
新建仓库名必须为：<用户名>.github.io
选择main分支作为部署分支
三、Hexo博客搭建
项目初始化
hexo init myblog # 创建博客项目
cd myblog && npm install # 安装依赖
本地预览
hexo g # 生成静态文件
hexo s # 启动本地服务器（http://localhost:4000）
四、自动化部署配置
修改站点配置
_config.yml
deploy:
type: git
repo: git@github.com:yourname/yourname.github.io.git
branch: main
部署到GitHub
npm install hexo-deployer-git –save # 安装部署插件
hexo clean && hexo g -d # 一键清理构建部署
五、内容管理
文章创作流程
hexo new “我的第一篇技术博客” # 创建Markdown模板
编辑source/_posts/我的第一篇技术博客.md文件
